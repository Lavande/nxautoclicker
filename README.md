# NX Auto Clicker

A modern desktop automation tool built with Tauri.

## Download

Go to the [Releases](https://github.com/Lavande/nxautoclicker/releases) page to download the latest version.

### Available platforms

| Platform | Format |
|----------|--------|
| Windows  | `.exe` (NSIS installer), `.msi` |
| macOS (Apple Silicon) | `.dmg` |
| macOS (Intel) | `.dmg` |
| Linux    | `.AppImage`, `.deb` |

## Installation

### Windows
Download and run the `.exe` or `.msi` installer.

### macOS
Download the `.dmg` file, open it, and drag the app to your Applications folder.

> **Note:** Since this app is not code-signed, you may need to right-click and select "Open" on first launch, or allow it in System Settings > Privacy & Security.

### Linux
- **AppImage:** Make it executable (`chmod +x *.AppImage`) and run it.
- **Deb:** Install with `sudo dpkg -i *.deb`.
